# Transmutation v1

The earliest versions of Transmutation focused on raw phase manipulation. By taking a standard sine wave and wrapping it through a series of mathematical folds, I was able to generate complex, metallic timbres that felt alive.

This post will eventually dive into the code behind the phase distortion algorithms and provide some audio examples of the very first hardware tests.
